# Contributing

Thank you for improving OpenVPN Control Agent.

1. **Issues** — Go version, OS, OpenVPN version, steps to reproduce.
2. **Pull requests** — one topic per PR; run `go test ./...`.
3. **Security** — see [SECURITY.md](SECURITY.md).

The **web panel** is a separate repository; do not commit server UI/API code here.
